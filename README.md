Tên đề tài: Mini game giới thiệu khóa học Robotics
Chức năng: Là một trò chơi giáo dục tương tác được xây dựng nhằm giới thiệu khoa học Robotics cho trẻ em. Trò chơi cho phép người chơi:
- Lựa chọn Robot: Chọn một trong ba mẫu robot khác nhau để bắt đầu lắp ráp.
- Lắp ráp tương tác: Trải nghiệm quy trình lắp ráp các bộ phận robot trên nền bản vẽ thiết kế (Design Plan).
Hướng dẫn chạy source code
Yêu cầu hệ thống
-  Đã cài đặt Python 3.x
-  Đã cài đặt thư viện pygame
Cài đặt thư viện pygame bằng cách mở terminal (hoặc Command Prompt) và chạy lệnh: pip install pygame
Khởi chạy trò chơi bằng cách di chuyển vào thư mục dự án và chạy tệp main.py
Ngôn ngữ và Công nghệ sử dụng:
-  Ngôn ngữ lập trình: Python.
-  Thư viện chính: pygame (sử dụng để xử lý đồ họa, âm thanh và sự kiện).
-  Định dạng dữ liệu: JSON (dùng để lưu trữ hệ thống câu hỏi trắc nghiệm).
